# yeah - 新时代笔记应用

一款极简但富有创新体验的笔记应用，灵感来自 2026 年最新笔记应用趋势。

## ✨ 特色功能

### 🤖 智能关联发现
- 写笔记时自动分析关键词
- 实时找出相关的旧笔记
- 帮助发现被遗忘的想法

### 🎨 极简设计
- 干净的纸质感界面
- 5 种主题色可选
- 专注内容本身

### 📱 流畅交互
- 反向时间线设计（最新笔记在底部）
- 长按探索关联
- 点击连接图标展开相关笔记

## 🚀 技术栈

- Flutter 3.x
- Material Design 3
- SharedPreferences 本地存储
- 纯 Dart 实现

## 📦 安装

```bash
flutter pub get
flutter run
```

## 🔨 构建 APK

```bash
flutter build apk --release
```

APK 文件位置: `build/app/outputs/flutter-apk/app-release.apk`
